This is my Bachelor's studies final year project named "Pharmacy automation system"
Details:
It is a Point of Sale for a pharmacy store developed on Dot Net platform which include following modules:
1. Login
2. Bill generation via bar code scanner
3. Stock maintenance and Low stock alerts
4. Attendance via bar code scanner
5. Attendance count or check

It was developed using ASP.NET platform using Microsoft Visual Studio 2010 and Microsoft SQL Server 2008. For front-end I used HTML, CSS, JavaScript and jQuery 
and C# as the back-end language.
